# Página de Apresentação

Projeto de portfólio pessoal desenvolvido com **HTML e CSS**, com o objetivo de apresentar informações profissionais, páginas de navegação e links para redes sociais.

O site reúne uma breve apresentação, informações sobre o autor e um resumo curricular, servindo como prática de desenvolvimento **front-end** e organização de páginas web.

---

## Funcionalidades

* Página inicial com apresentação pessoal
* Página "Sobre mim" com descrição e objetivos profissionais
* Página "Currículo" com formação acadêmica e cursos realizados
* Menu de navegação entre páginas
* Links para redes sociais (LinkedIn, GitHub e Instagram)
* Layout responsivo utilizando Flexbox

---

## Como Executar o Projeto

1. Clone o repositório

git clone https://github.com/GabrielLengertGuedes/Pagina-de-apresentacao.git

2. Acesse a pasta do projeto

cd Pagina-de-apresentacao

3. Abra o arquivo **index.html** em qualquer navegador.
